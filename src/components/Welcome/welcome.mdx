# eteui(Eternity-UI)

一个基于 React+Typescript 的 UI 组件库，意在成为让开发者能快速成型网站的工具。

## 安装

```js
npm install eteui
```

## 快速上手

```js
// 加载样式
import "eteui/dist/index.css";
// 引入组件
import { Button } from "eteui";
```

## 组件库亮点

- 🌹 组件丰富，选用自如
- 🔥typescript with React Hooks
- 📦 开箱即用
- 🔐 组件单元测试覆盖率高
